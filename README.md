React Bootstrap Webpack starter
==========

A simple ReactJS + Bootstrap starter.
Bundler is the commonly used by react community  : `webpack`.

## Detailed Content

- React JS
- react-router
- Bootstrap
- React-Bootstrap
- Webpack
- animate.css
- classnames
- font-awesome



## Usage

### Install

```bash
npm install
```
### bundle dev mode

```bash
npm run dev
```

### dev : hot reload mode

```bash
npm run start
```

### bundle production mode

```bash
npm run prod
```
